**3.IQR을 계산하기 위한 Q1, Q3 계산**

```python
# np.percentile(열, 퍼센트)
Q1=np.percentile(titanic['Fare'],25)
Q3=np.percentile(titanic['Fare'],75)
Q1, Q3       # 데이터 확인
```




    (7.9104, 31.0)



**3. IQR 계산**

IQR(사분범위) = 제3사분위수(75%) - 제1사분위수(25%)


```python
IQR = Q3-Q1
```

**4. 이상치 데이터를 판별하기 위한 기준 설정**

```python
#cut_off 정의
btom_cut_off = Q1 - 1.5*IQR  #하한
top_cut_off = Q3 + 1.5*IQR   #상한
btom_cut_off, top_cut_off    #데이터 확인
```




    (-26.724, 65.6344)



**5. outlier 변수에 이상치 데이터 저장**

```python
outlier=titanic[(titanic['Fare'] <= btom_cut_off) | (titanic['Fare'] >=top_cut_off)]
  #해석: outlier 변수에 titanic 데이터를 저장할건데, 인덱스는 'fare'변수의 값이           btom_cut_off보다 작거나(or, |) top_cut_off 보다 큰 값으로.
  
outlier   #데이터 확인
```
